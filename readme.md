# TILTBOT


## Description

Tiltbot is a telegram bot using LoL API to analyze asked game and very non scientifically determine how mad a player probably is about their last game.

Technologies used:

 - Flask
 - Telegram API
 - Docker
 - Riot Games API
 - Redis
